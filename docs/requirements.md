需求说明书

项目名称：基于 ESP32-C3 的番茄钟与智能聊天助手设备

概述
- 目标平台：ESP32-C3
- 显示：ST7789 240x240 LCD，通过 SPI 驱动
- UI：LVGL 图形库，支持导航、设置、时间等信息显示
- 输入：EC11 编码器，旋转用于导航，按键进入/确认
- 网络：WiFi 连接管理，NTP 时间同步

---

功能性需求

1. 界面系统
   - 主界面：显示本地时间（年月日时分秒）、温湿度、天气、WiFi 状态
   - 番茄钟界面：倒计时状态（分钟秒）、完成次数
   - 聊天助手界面：助手表情、对话内容
   - 设置界面：WiFi 设置、番茄钟设置、聊天助手设置、时区和地区设置
   - 设置子界面：WiFi 扫描列表、密码输入、番茄钟时间设置等

2. WiFi 功能
   - WiFi 配网：扫描周围可用网络、显示列表、选择网络、输入密码、连接
   - 连接状态显示
   - NTP 时间同步，时区配置
   - 天气查询

3. 番茄钟功能
   - 可配置的工作时长与休息时长
   - 倒计时显示（分钟:秒）
   - 完成次数记录与显示
   - 支持开始、暂停、继续、重置、清空次数操作
   - 番茄钟设置子界面：工作时间设置、休息时间设置、清空完成次数

4. 输入控制
   - EC11 编码器：
     - 旋转：界面间前后循环切换 / 设置项滚动
     - 按键：回退功能
   - 独立设置键（GPIO9）：
     - 按下：进入设置界面 / 触发对应界面功能（开始番茄钟、开始聊天等）

5. 离线工作能力
   - 网络异常时番茄钟功能仍可独立运行
   - 显示离线状态但保留本地时钟

---

非功能性需求

1. 性能与稳定性
   - LVGL 主循环与任务调度的合理优先级
   - DMA/硬件 SPI 优化数据传输
   - UI 响应性：页面刷新应在 100-200ms 内完成

2. 安全与容错
   - 连接失败时提供可观测提示
   - 输入消抖与按键去抖处理
   - WiFi 配网信息在 NVS 中持久化

3. 资源限制
   - RAM/Flash 使用需在目标设备可承受范围内
   - 定时清理无用缓存

4. 兼容性
   - 支持 240x240 分辨率，易扩展到其他分辨率
   - 模块化架构便于硬件替换

---

硬件约束

- MCU：ESP32-C3

- 引脚分配：
  | GPIO  | 功能            | 说明                      |
  |-------|-----------------|---------------------------|
  | GPIO0 | I2S WS / LRCK  | 麦克风与功放共用          |
  | GPIO1 | I2S BCLK       | 麦克风与功放共用          |
  | GPIO2 | I2S SD (IN)    | 麦克风数据输入            |
  | GPIO3 | I2S DOUT (OUT) | 功放数据输出              |
  | GPIO4 | Encoder A      | EC11 A相                  |
  | GPIO5 | Encoder B      | EC11 B相                  |
  | GPIO6 | SPI SCL        | 显示屏时钟                |
  | GPIO7 | SPI SDA        | 显示屏数据                |
  | GPIO9 | Button         | 独立设置键                |
  | GPIO10| SPI CS         | 显示屏片选                |
  | GPIO21| Encoder Key    | EC11 按键                 |

- 外设：ST7789 LCD、EC11 编码器、INMP441 麦克风、MAX98357 功放

---

术语表
- LVGL：Light and Versatile Graphics Library，嵌入式 GUI 框架
- ST7789：彩色 LCD 控制器，SPI 总线
- EC11：编码器，带按键
- NTP：网络时间协议，用于时间同步
- NVS：Non-Volatile Storage，非易失性存储
- I2S：Inter-IC Sound，数字音频接口
